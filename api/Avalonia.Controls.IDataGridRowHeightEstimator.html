Interface IDataGridRowHeightEstimator

Namespace
Avalonia.Controls
Assembly
Avalonia.Controls.DataGrid.dll

Defines the contract for row height estimation and scroll position calculation in DataGrid. Implementations can provide different algorithms for handling variable row heights.
