AdvancedRowHeightEstimator

Advanced implementation of IDataGridRowHeightEstimator that provides optimized handling for variable row heights. Features include:

  • Height cache for measured rows
  • Regional averages for localized estimation
  • Smooth scroll correction
  • Fenwick tree for efficient cumulative height queries
  • Bidirectional estimation updates
CachingRowHeightEstimator

Advanced implementation of IDataGridRowHeightEstimator that caches individual row heights for more accurate scrolling with variable height rows.
